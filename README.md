## Описание проекта:
>Проект направлен на повышение эффективности работы интернет‑магазина за счёт предиктивной аналитики поведения пользователей.
>На основе исторических данных о взаимодействиях клиентов с ресурсом необходимо построить модель, которая будет прогнозировать вероятность совершения покупки в ближайшие 90 дней.
>Это позволит:
+целенаправленно формировать персонализированные предложения;
+оптимизировать бюджет на маркетинговые рассылки;
+повысить конверсию за счёт фокусировки на «горячей» аудитории.

## Цель проекта
>Разработать модель машинного обучения, способную с максимальной точностью предсказывать вероятность того, что конкретный пользователь совершит покупку в течение 90 дней с момента анализа.
>Ключевая метрика качества - ROC‑AUC.
